Connect AutomatorWP with Restrict Content Pro

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[Restrict Content Pro](https://restrictcontentpro.com/ "Restrict Content Pro") is one of the most featured and powerful membership solutions for WordPress that lets you lock away your exclusive content.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with Restrict Content Pro adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User gets any/specific free membership level.
* User purchases any/specific membership level.
* User cancels a subscription of any/specific membership level.

= Actions =

* Add membership level to the user.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/restrict-content-pro/ "AutomatorWP - Restrict Content Pro") available that let's you expand the integration between AutomatorWP and Restrict Content Pro bringing you more ways to automatize your website forums.

= What is AutomatorWP? =

AutomatorWP is an automator plugin that lets you connect your WordPress plugins together and add automation to unlimited workflows.

You can automate anything including sales, marketing, administrative tasks, learning and any other kind of processes you want letting you save time and get focused on your most important work.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
